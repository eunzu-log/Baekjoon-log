## 📝 문제 번호 1085 - 직사각형에서 탈출

### 📌 문제
한수는 지금 (x, y)에 있다. 직사각형은 각 변이 좌표축에 평행하고, 왼쪽 아래 꼭짓점은 (0, 0), 오른쪽 위 꼭짓점은 (w, h)에 있다. 직사각형의 경계선까지 가는 거리의 최솟값을 구하는 프로그램을 작성하시오.

---

### 📥 입력
첫째 줄에 x, y, w, h가 주어진다.

---

### 📤 출력
첫째 줄에 문제의 정답을 출력한다.

---

### ⛔ 제한  
- 1 ≤ w, h ≤ 1,000
- 1 ≤ x ≤ w-1
- 1 ≤ y ≤ h-1
- x, y, w, h는 정수

---

### 💻 코드
```c
#include <math.h>
#include <stdlib.h>
#include <string.h>
#include <stdio.h>

int main() {
	int x, y, w, h, dist[4] = { 0 }, min = 1000;
	scanf("%d %d %d %d", &x, &y, &w, &h);
	dist[0] = x;
	dist[1] = y;
	dist[2] = w - x;
	dist[3] = h - y;
	for (int i = 0; i < 4; i++) {
		if (dist[i] < min) min = dist[i];
	}
	printf("%d", min);
}
```
