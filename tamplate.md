## 📝 문제 번호 - 문제 제목

### 📌 문제
> 문제 설명을 간단히 요약하거나 붙여넣기  
> (예: 두 정수 A와 B를 입력받은 다음 A+B를 출력하는 문제)

---

### 📥 입력
> 입력 조건 정리  
> (예: 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10))

---

### 📤 출력
> 출력 조건 정리  
> (예: 첫째 줄에 A+B를 출력한다.)

---

### 💻 코드
```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a + b);
    return 0;
}
```
